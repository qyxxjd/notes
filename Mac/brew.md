#### 命令帮助

``` 
brew --help 
```

#### 查看已安装的软件列表

``` 
brew list 
```

#### 安装、卸载

``` 
brew install ***
brew uninstall ***
```

#### 更新

``` 
brew update      // 查看已安装软件版本更新信息
brew outdated    // 查看需要更新的软件
brew upgrade *** // 更新某个软件
```

#### 搜索

``` 
brew search *** 
```

#### 查看软件信息

``` 
brew info *** 
```

#### 清理软件的旧版本

``` 
brew cleanup     // 清理所有旧版本
brew cleanup *** // 清理某个软件的旧版本
```
